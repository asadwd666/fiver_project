CondoSites is coded with modern features that are not supported by Internet Explorer.
In order to use this site you need to upgrade your browser to either
Google Chrome, Microsoft Edge, Apple Safari, or Mozilla Firefox.

What is a browser?
A web browser is a software program that allows a user to locate, access, and display web pages. The most common browsers are Chrome by Google, Edge by Microsoft, Safari by Apple, and Firefox by Mozilla.

Why can't I view this site on Internet Explorer?
Microsoft discontinued Internet Explorer in January of 2016 and replaced it with Edge. The most recent version (Internet Explorer 11) lacks adoption of many HTML5 and CSS web standards. Features coded into CondoSites do utilize these modern standards and we plan to use more of them. So to make sure everyone has the same excellent experience, and to offer better functionality in the future, we have adopted certain minimum browser requirements.

But everyone uses Internet Explorer?
Not true anymore. While Internet Explorer once had 90% market share, today it has less than 3%. (Source: Wikipedia - January 22, 2019)

What should I do?
In order to use this site, and many others, you need to upgrade your browser to either Google Chrome, Microsoft Edge, Apple Safari, or Mozilla Firefox. All are free to download and use.

Which browser is best?
Each browser is a little different, so the choice is up to you. The most commonly used browser for our software is Google Chrome, and we develop our platform on Mozilla Firefox.
